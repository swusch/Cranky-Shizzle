# Texture Placeholders

The following texture files need to be created (16×16 PNG):

## Block Textures
- `assets/crankycolonists/textures/block/metallurgist_hut.png`
  - Suggested: Dark iron/smithing table inspired texture

## Item Textures
- `assets/crankycolonists/textures/item/creative_icon.png`
  - Suggested: Hammer/anvil icon for the creative tab

## Citizen Textures (128×64 PNG)
- `assets/crankycolonists/textures/entity/metallurgist_male.png`
- `assets/crankycolonists/textures/entity/metallurgist_female.png`

## Notes
- The block model currently references vanilla textures (smithing_table, iron_block, anvil, redstone_block, smooth_stone) 
  so no custom block textures are strictly required for the block to render.
- The creative_icon item model references `crankycolonists:item/creative_icon` and DOES need a texture file.
- You can use any vanilla texture as a temporary placeholder by copying it to the above paths.
